# Python PlataformGame
### Teste para tutores - Python

#### Para iniciar o jogo primeiro clone o repositorio em uma pasta
- usando SSH
    - `git clone git@github.com:Fkaique/python_PlataformGame.git` 
- usando HTTP
    - `git clone https://github.com/Fkaique/python_PlataformGame.git`
#### Apos a criação dos arquivos entre na pasta do projeto, crie um ambiente virtual venv e instale as dependencias
- criando o ambiente virtual
    - `python -m venv venv`
- iniciando-o com Linux/MacOS
    - `source venv/bin/activate`
- iniciando-o com Windows
    - `venv\Scripts\activate`
- instalando as dependencias no projeto
    - `pip install -r requirements.txt`
